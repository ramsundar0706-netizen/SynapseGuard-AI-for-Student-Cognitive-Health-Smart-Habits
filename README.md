# 🧠 SynapseGuard
### AI for Student Cognitive Health & Smart Habits
> Health & Wellbeing Hackathon Project

---

## 🚀 What It Does
SynapseGuard analyses your sleep, screen time, stress, mood & air quality to generate a personalised AI cognitive health report.

---

## 🛠️ Tech Stack
| Frontend | Backend | API |
|----------|---------|-----|
| React + Vite | Python Flask | Open-Meteo AQI |

---

## ⚡ Quick Start

**Backend**
```bash
cd backend
py -m pip install -r requirements.txt
py app.py
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
Open → `http://localhost:3000`

---

## ✨ Features
- 📋 5-step health assessment
- 🤖 AI scoring engine
- 🌍 Live AQI integration
- 📊 Visual health report
- 🧠 Depression risk indicator
