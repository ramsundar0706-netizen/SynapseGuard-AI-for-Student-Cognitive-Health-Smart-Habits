# SynapseGuard
AI for Student Cognitive Health & Smart Habits

## Problem
Students often experience burnout, cognitive fatigue, and poor study habits due to long study hours and lack of healthy routines.

## Solution
SynapseGuard uses AI to analyze study behavior and detect early signs of cognitive stress, providing smart micro-habit recommendations to maintain focus and mental well-being.

## Features
- Cognitive fatigue detection
- Burnout risk prediction
- Smart micro-habit suggestions
- Health insights dashboard

## Tech Stack
Frontend: React + Tailwind  
Backend: Node.js + Express  
AI: Python (ML models)
